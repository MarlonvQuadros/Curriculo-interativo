Sobre o Projeto

Este é o meu primeiro projeto pessoal, onde utilizei meus conhecimentos acadêmicos adquiridos no curso de Análise e Desenvolvimento de Sistemas, bem como os conhecimentos que aprendi no curso de HTML e CSS do "Curso em Vídeo". Durante o desenvolvimento do site, busquei aplicar boas práticas de design e usabilidade, criando uma interface simples e funcional para apresentar minhas informações de maneira clara e objetiva.

Além disso, utilizei o ChatGPT para adaptar algumas funcionalidades do site, como a organização e a estruturação do conteúdo, o que me permitiu otimizar o processo e aprender novas técnicas para melhorar ainda mais a experiência do usuário.
